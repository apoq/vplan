# VPlan App
======================

## Info

This is a sample app built with custom made framework, which contains:

 * nezamy/route router
 * twig templating engine
 * propel orm

## Setup

1) Clone the project
```
$ git clone http://github.com/apoq/vplan
```

2) Move to project root
```
$ cd vplan/
```

3) Configure project
```
$ nano/vim propel.yml
```

4) Build project
```
$ php setup.php
```

5) Run
```
$ cd public/ && php -S 0.0.0.0:8080
```
