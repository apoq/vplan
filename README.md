# API Client Example
======================

## Setup

1) Clone the project
```
$ git clone http://github.com/apoq/vplan
```

2) Move to project root
```
$ cd vplan/
```

3) Configure project
```
$ nano/vim propel.yml
```

3) Build project
```
$ php setup.php
```

4) Run
```
$ cd public/ && php -S 0.0.0.0:8080
```
